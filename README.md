# SISTEMA-WEB-PORTABLE-DE-AGENDAMIENTO-CLINICO-E-HISTORIAS-CLINICAS
Sistema web portable (Zero-Configuration) con persistencia en SQLite para el agendamiento de citas médicas por colas FIFO e historias clínicas dinámicas adaptativas por especialidad. Desarrollado bajo enfoque IHC para el Trabajo Práctico Experimental 4 - UNEMI.
